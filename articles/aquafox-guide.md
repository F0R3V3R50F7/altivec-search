## What Is Aquafox?

Aquafox is a modern Firefox-based browser built specifically for PowerPC Macs. It's maintained by BlackBirdLC and is currently the best browser option for G4 hardware. It handles modern TLS, renders most sites properly, and runs well on limited resources.

## Which Version Do I Need?

There are two builds depending on your G4 processor:

- **7450** — for Quicksilver, Mirror Drive Door (MDD), and FireWire 800 PowerMacs. These have the later G4 chip with improved AltiVec.
- **7400** — for AGP Graphics, Gigabit Ethernet, and Digital Audio PowerMacs. These use the original G4 chip.

Not sure which you have? Open System Profiler (in Applications > Utilities) and check the CPU type.

## Installation

Download the correct DMG from the links in the sidebar. GitHub won't load in older browsers, so use the direct download links provided here — they'll work fine.

1. Download the .dmg file
2. Double-click to mount it
3. Drag Aquafox to your Applications folder
4. Launch Aquafox from Applications

That's it. No installer, no configuration, no dependencies.

## Tips

- Set Aquafox as your default browser in System Preferences > Internet & Network
- Bookmark [altivec.fyi](https://www.altivec.fyi) as your homepage for quick access to G4 resources
- If sites load slowly, check the connection — Aquafox itself is not the bottleneck on most pages
